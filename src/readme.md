# gz

About gulp's configuration, one click to avoid gulpfile files that are always duplicated.
(关于gulp的配置一键化，避免总是重复的配置gulpfile文件)

## 支持的写法

* babel Es6/7

* autoprefixer 

* px/rem (px转rem)

* [gulp-file-include](https://www.npmjs.com/package/gulp-file-include) template

```
$ npm install -g gulp

$ npm install

$ gulp
```
